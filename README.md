# Sistem Kasir Sederhana (PBO)

## Deskripsi
Proyek ini merupakan aplikasi **Sistem Kasir Sederhana** berbasis Python
yang dibuat untuk memenuhi tugas mata kuliah **Pemrograman Berorientasi Objek (PBO)**.

Aplikasi ini menerapkan konsep dasar PBO seperti:
- Class
- Object
- Method
- Constructor

## Fitur
- Menambahkan data barang
- Menampilkan daftar belanja
- Menghitung total pembayaran
- Menampilkan struk belanja sederhana
